## 游戏概述

在神话贪吃蛇中，你将控制一条不断成长的蛇，在充满挑战的世界中生存、战斗、进化。击败强大的Boss获取传说装备，收集卡牌强化能力，探索神秘的环境。

---

## 基础操作

| 按键 | 功能 |
|-----|------|
| **方向键 / WASD** | 控制移动方向 |
| **Shift** | 加速移动（消耗能量） |
| **Space** | 闪现（需解锁） |
| **Z** | 时间回溯（最多3次） |
| **ESC** | 暂停游戏 |
| **I** | 打开/关闭仓库 |
| **C** | 打开/关闭卡牌面板 |

## 核心系统

### 波次系统

游戏以波次形式推进，每25秒进入下一波：

| 波次 | 事件 |
|-----|------|
| 每1波 | 敌人数量+1，难度提升10% |
| 每3波 | **Boss战** + 卡牌抽取 |
| 每5波 | 精英敌人出现概率+15% |

### 能量系统

- **能量上限**: 100点
- **恢复方式**: 静止状态、食用食物
- **消耗方式**: 
  - 普通移动: 0.1/秒
  - 加速移动: 0.5/秒
  - 闪现: 25点/次
  - 发射子弹: 0.25点/次

### 生命值系统

- 初始生命: 100点
- 受到伤害后会短暂无敌
- 生命回复: 某些装备提供持续回复

## Boss系统

### Boss出现规则

- 每3波出现一个Boss
- Boss出现前有5秒警告时间
- 击败Boss后获得掉落物

### Boss类型

| Boss | 名称 | 特点 | 难度 |
|------|------|------|------|
| 🐍 | 九头蛇 | 多头攻击，生命回复 | ★★★ |
| 🔥 | 炎魔 | 火焰攻击，范围伤害 | ★★★★ |
| 🛡️ | 泰坦 | 高防御，重击 | ★★★★ |
| ⚡ | 雷神 | 闪电链，高机动 | ★★★★★ |
| 👑 | 混沌之王 | 混乱攻击，多阶段 | ★★★★★★ |

### Boss战斗技巧

1. **观察攻击模式** - 每个Boss有固定的攻击节奏
2. **利用闪现躲避** - 关键时刻使用闪现逃脱
3. **保持距离** - 远程攻击更安全
4. **注意阶段变化** - Boss血量降低后会进入狂暴状态

## 新手攻略

### 第一步：熟悉操作

1. 使用方向键控制蛇的移动
2. 按住Shift可以加速，但会消耗能量
3. 学会使用闪现（Space）躲避危险

### 第二步：收集资源

1. 吃掉地图上的食物来增长身体
2. 注意能量条，不要过度消耗
3. 收集特殊食物来进化器官

### 第三步：应对敌人

1. 保持移动，不要停留在原地
2. 利用子弹远程攻击敌人
3. 遇到危险时使用闪现逃脱

### 第四步：挑战Boss

1. 每3波会出现Boss
2. Boss出现前有5秒警告
3. 击败Boss获得稀有装备
4. 在仓库中装备获得的物品

### 第五步：强化角色

1. 每3波选择一张卡牌
2. 在仓库中管理装备
3. 合理搭配饰品和武器
4. 利用Buff效果增强战斗力

### 进阶技巧

1. **走位风骚**: 利用蛇身的灵活性躲避攻击
2. **资源管理**: 合理分配能量使用
3. **装备搭配**: 根据Boss类型选择合适装备
4. **时间回溯**: 关键时刻使用回溯保命
5. **环境利用**: 利用地形和虫洞获得优势

## 常见问题

**Q: Boss不出现怎么办？**
A: Boss每3波出现一次，请耐心等待波次推进。

**Q: 装备后武器没有效果？**
A: 确保在仓库中正确装备武器，武器会替换默认攻击。

**Q: 如何获得更好的装备？**
A: 击败更高难度的Boss，掉落稀有度更高的装备。

**Q: 卡牌选择有什么技巧？**
A: 根据当前游戏风格选择，攻击型选伤害，生存型选防御。
